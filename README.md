# 🚀 CSE Hackathon Platform – BVCOE Delhi

An AI-enabled digital platform developed for managing hackathon workflows including idea submission, domain-wise categorization, and innovation tracking.

---

## 🎯 Context of Use

This platform is currently designed and deployed to support **Hack@BVP 7.0**, enabling structured participation, idea submission, and tracking of student innovation activities.

---

## 🎯 Key Features

* Idea submission system for student teams
* Domain-based categorization (AI, Cybersecurity, Cloud, Web, Blockchain)
* Live submission dashboard
* Structured innovation pipeline tracking

---

## 🧠 Academic & Innovation Impact

This platform is designed to strengthen:

* Project-Based Learning (PBL) implementation
* Student innovation and problem-solving culture
* Research and publication pipeline development
* Startup and incubation ecosystem alignment

---

## 🔥 Outcome Path

Hackathon → Final Year Project → Research Paper → Startup

---

## 🛠️ Technology Stack

* Streamlit (Frontend + Backend)
* Python
* CSV-based lightweight data storage
